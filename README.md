# Vista Motors Customer Support Chatbot

A Prolog-based expert system for Vista Motors motorcycle dealership that helps customers check:
- Bike prices
- Bike categories/types
- Stock availability

## Features
- Natural language query processing
- Interactive chat interface
- Knowledge base of 8 motorcycle models
- Real-time stock status checking

## Requirements
- SWI-Prolog (https://www.swi-prolog.org/)

## How to Run
1. Install SWI-Prolog
2. Load the program:
   ```prolog
   ?- [vista_motors].
